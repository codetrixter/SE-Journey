# Networking Concepts Curriculum

## Course Introduction
- Get introduced to the course and its syllabus, setting the foundation for your learning journey.
- The course's introductory video will provide insights into the valuable skills and knowledge you can expect to gain throughout the duration of this course.

## Introduction to Networking and Internet
- Overview of current computer networks and the internet, their evolution, and uses.
- What is a protocol and why do we need it?
- Entities involved: edge, access networks, core side.
- Key characteristics influencing network performance: throughput, delay.
- Layered architecture: OSI and TCP/IP models.

## Internet Applications and Communications
- Principles of network applications.
- Client-server vs. peer-to-peer architecture.
- World Wide Web (WWW), HTTP protocol, Domain Name System (DNS), Email protocols.

## Transport Layer Protocols
- Major transport layer protocols: UDP and TCP.
- Multiplexing and demultiplexing.
- Reliable data transfer, flow control, connection management, congestion control.

## IP Network – Addressing and Data Transfer
- Network layer services and packet switching.
- Internet Protocol version 4 (IPv4): addressing, packet format, options, ICMPv4, mobile IP.
- Introduction to IP version 6 (IPv6).

## Routing – Control Plane and Data Plane
- Router forwarding tables and packet forwarding.
- Routing protocols.
- Network layer decomposition: data plane and control plane.
- Multicasting concepts, SDN approach.

## Multimedia Transport
- Generation, storage, and transmission of integrated media: text, images, audio, video.
- Compression and digitising multimedia data.
- Multimedia protocols and real-time conversational application protocols.

## Data Link Layer and LAN
- Role and functioning of the link layer.
- Error detection and correction mechanisms.
- Multiple access problem coordination mechanisms.
- Ethernet: the most widely used LAN technology.

## Physical Layer and Wireless Networks
- Analogue and digital data/signals and their characteristics.
- Conversion: digital data to digital signals, analogue data to digital signals.
- Multiplexing for bandwidth efficiency.
- Transmission media and wireless/mobile networks.

## Network Management
- Key functionalities: Fault, Configuration, Accounting, Performance, Security (FCAPS).
- Simple Network Management Protocol (SNMP): framework, operations, significance.
- Role of NETCONF and YANG in network management.

## Network Security
- Principles of securing network applications.
- Transport Layer Security (TLS): securing HTTP applications, connection establishment, handshaking, key derivation, safe data transfer.
- Security of Email communication.
- Firewalls and Intrusion Detection Systems (IDS) for operational security.
