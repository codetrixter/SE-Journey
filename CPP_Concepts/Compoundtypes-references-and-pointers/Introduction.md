# COMPOUND TYPES IN C++

## Every data type in C++ is either a fundamental data type or a compound data type.

## C++ Compound Types

C++ supports the following compound types:

- **Functions**
- **C-style Arrays**
- **Pointer types**
  - Pointer to object  
  - Pointer to function  
- **Pointer to member types**
  - Pointer to data member  
  - Pointer to member function  
- **Reference types**
  - L-value references  
  - R-value references  
- **Enumerated types**
  - Unscoped enumerations  
  - Scoped enumerations  
- **Class types**
  - Structs  
  - Classes  
  - Unions
